# App de Notas

Aplicacion responsive para crear, organizar y conservar notas locales con React,
TypeScript y persistencia en el navegador.

## Estado

Estructura inicial creada:

- proyecto React con TypeScript y Vite;
- Tailwind CSS configurado;
- base preparada para GitHub Pages;
- pantalla inicial responsive;
- documentacion tecnica inicial;
- estructura prevista para modelado SQL.

## Stack

- React;
- TypeScript;
- Vite;
- Tailwind CSS;
- Lucide React;
- localStorage;
- SQL documentado;
- Git y GitHub.
